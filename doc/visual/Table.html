Classpublic class Table
InheritanceTable flash.display.Sprite

Table contains two columes, the left colume contains name cells and the right colume contains value cells. A table represent a JSON object in design and capable to generate the preview drawing result for different type of data structure defined in SWF (such as Edge, Shape, Gradient, etc...).
